# National Weather ServiceThis is a repo for managing machine readable artifacts for the [National Weather Service APIs](http://weather.gov). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [National Weather Service APIs](http://weather.gov).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://weather.gov)- [Developer](http://weather.gov)- [Twitter](https://twitter.com/NWS)The [APIs.json](https://github.com/api-evangelist/national-weather-service/blob/master/apis.json) index provides links to all the relevant links for the National Weather Service APIs, including the OpenAPI and Postman if they are present.
